atom-jsfmt
==========

Automatically run `jsfmt` every time you save a javascript source file!


Usage
-----


Make sure you have `jsfmt` installed:

```bash
npm install -g jsfmt
```

If you have a custom location for the `jsfmt` binary, set the `pathToJsfmt`
option accordingly.
